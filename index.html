Atajos esenciales

  • /Inserta bloques con vista previa
  • Ctrl/Cmd + FAbre buscar y reemplazar
  • Ctrl/Cmd + Shift + HAbre buscar y reemplazar desde cualquier vista
  • Ctrl/Cmd + BNegrita sobre la seleccion
  • Ctrl/Cmd + ICursiva sobre la seleccion
  • Ctrl/Cmd + KCrea un enlace
  • Ctrl/Cmd + Alt + NAgrega o edita una nota interna del bloque actual
  • Ctrl/Cmd + Shift + FActiva modo foco
  • Ctrl/Cmd + .Abre o cierra el inspector
  • Ctrl/Cmd + ,Abre preferencias
  • F1Abre esta guia
  • EscCierra menus, guia o foco
